## 🚀 Run Project

1️⃣ Clone project and access its folder:

```bash
$ git clone https://github.com/jtiagosantos/GoRestaurant.git
$ cd GoRestaurant
```

2️⃣ Install dependencies:

```bash
$ yarn or npm i
```

3️⃣ Start project:

```bash
$ yarn server or npm run server
$ yarn dev or npm run dev
```

4️⃣ Run tests:

```bash
$ yarn test or npm test
```
